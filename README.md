## Objetivo

Aplicações em React Native para a matéria de Linguagem de Programação da faculdade de Sistemas de Informação.

<br>

### Projetos

1. Calcular área de um triângulo.
2. Mostrar longitude e latitude a partir do acesso a localização do celular.
3. Pesquisar e mostrar contatos a partir do acesso a lista de contatos do celular.

<br>

### Tecnologias

- [React Native](https://reactnative.dev/)
- [Expo](https://expo.io/)

<br>

### O código

Os projetos estão em 3 pastas diferentes e todos os códigos se concentram nos arquivos `App.js` respectivos de cada pasta.
Para rodar será necessário o uso do [Expo instalado na máquina](https://expo.io/) e caso venha a ser testado no próprio celular, será necessário [instalar no celular](https://play.google.com/store/apps/details?id=host.exp.exponent&hl=pt_BR) também. _Ou utilizar um simulador_
- [Como funciona o Expo com o React Native](https://medium.com/@rogercruz/react-native-com-expo-um-exemplo-40e5574c6904);

<br> 

### Passo a passo
_(Necessário ter o [node](https://nodejs.org/en/), [yarn](https://yarnpkg.com/) ou [npm](https://www.npmjs.com/) instalados na máquina)_
1. Clonar o projeto em sua máquina
2. Entrar em um dos projetos (pastas) e através do terminal instalar as dependências com `yarn install` ou `npm install`
2. Após a instalação `yarn start` para rodar a aplicação
